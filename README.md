# How to install
## Install web framework and install web project
- install nodejs  
https://nodejs.org/en/download/

- Command line  
    + `cd apartment-price-estimating`  
    + `npm install`

## Install python for prediction
- install python
- install numpy, joblib, sklearn by using the following commands:
    + `pip install numpy`
    + `pip install joblib`
    + `pip install sklearn`

# How to run
- Command line  
`npm start`

- Open on browser  
http://localhost:3001/
